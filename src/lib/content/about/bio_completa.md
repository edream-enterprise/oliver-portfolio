# Oliver's Bio

Welcome to my full biography. I am a passionate software engineer focused on building robust and elegant solutions from the bare metal to the DOM.

## My Journey
- **2020:** Discovered the terminal. The simplicity and power of the command line changed how I viewed computing forever.
- **2021:** Transitioned entirely to Linux (Fedora/Alma). Started scripting and automating my workflows.
- **2022:** Embraced Neovim & Hyprland for maximum efficiency. My development environment became a highly customized tiling window manager.
- **Present:** Architecting solutions. I now build complex web applications using SvelteKit, always keeping performance and aesthetics in mind.

## Philosophy
Simplicity is the ultimate sophistication. I prefer tools that do one thing well. The UNIX philosophy strongly influences my approach to software architecture. I believe in writing code that is not just functional, but also beautiful and easy to maintain.

## Skills
- **Languages:** TypeScript, JavaScript, HTML, CSS, Bash
- **Frameworks:** SvelteKit, Tailwind CSS
- **Tools:** Neovim, Git, Podman, Linux
